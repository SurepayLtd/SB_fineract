  USAGE FLOW
================================================================================

STEP 1: BUILD & DEPLOY
  Action: ./gradlew clean build
  Result: Required migration runs automatically
  Effect: two_factor_enabled column added to tenants table

STEP 2: CONFIGURE TENANTS (Choose one method)

  Method A: Liquibase Changesets (Recommended)
    liquibase --contexts=enable-2fa-production update

  Method B: Direct SQL
    UPDATE tenants SET two_factor_enabled = true WHERE identifier = 'production';

STEP 3: VERIFY
  SELECT identifier, two_factor_enabled FROM tenants;

STEP 4: TEST
  Login to configured tenant → Should prompt for 2FA
