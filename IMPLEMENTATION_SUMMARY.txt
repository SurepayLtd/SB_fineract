================================================================================
  HOW IT WORKS
================================================================================

2FA is enforced when ALL THREE conditions are met:

   1. Global flag is TRUE: fineract.security.2fa.enabled=true
   2. Tenant flag is TRUE: tenants.two_factor_enabled=true
   3. User does NOT have: BYPASS_TWO_FACTOR_PERMISSION

Logic Flow:
   User Login
      ↓
   Check fineract.security.2fa.enabled
      ↓ (if false) → No 2FA
      ↓ (if true)
   Check tenant.two_factor_enabled
      ↓ (if false) → No 2FA
      ↓ (if true)
   Check user bypass permission
      ↓ (if has permission) → No 2FA
      ↓ (if no permission) → REQUIRE 2FA
